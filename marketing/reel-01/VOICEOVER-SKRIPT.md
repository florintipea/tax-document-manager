# Voiceover für Reel #1 (optional)

**Gesamtdauer:** ca. 28 Sekunden (passt zu den 5 Slides)

---

## Kostenlose TTS-Tools (deutsch)

| Tool | Link | Hinweis |
|------|------|---------|
| **CapCut** (eingebaut) | In der App: Text → „Text vorlesen“ | Einfachste Option, schon installiert |
| **ElevenLabs** | elevenlabs.io | 10.000 Zeichen/Monat gratis, sehr natürlich |
| **Narakeet** | narakeet.com | Deutsch, Web, gratis mit Limit |
| **Google Translate** | translate.google.com → Lautsprecher | Schnell zum Testen, Qualität ok |

**Empfehlung für Anfänger:** CapCut Text-to-Speech direkt im Reel-Projekt.

| Zeit | Slide | Text |
|------|-------|------|
| 0:00–0:02 | 1 | „Deine Steuerunterlagen sehen so aus?“ |

| 0:27–0:30 | 5 | „Kommentier BETA, wenn du Fragen hast.“ |

## Volltext (zum Kopieren in TTS)

```
Deine Steuerunterlagen sehen so aus?

Jahr für Jahr derselbe Stress.

Ich hab TaxDoc gebaut: Dokumente hochladen, KI sortiert, Rechner schätzt deine Steuer.

Die Beta ist kostenlos — Link in Bio.

Kommentier BETA, wenn du Fragen hast.
```

---

## CapCut: Text vorlesen (3 Schritte)

1. **Text** → Text eingeben (Abschnitt für Abschnitt)
2. Text markieren → **„Text vorlesen“** / „Text to speech“
3. Stimme: **Deutsch** wählen (z. B. „Deutsch — weiblich“)

Jeden Abschnitt an die passende Slide-Position ziehen (siehe Zeitmarken oben).
